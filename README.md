 #Weather App

A simple weather application that allows users to search for current weather information by city name.

## Features

- Search weather by city name
- Display current temperature in Celsius
- Show weather description and icon
- Responsive design with background image
- Error handling for invalid inputs

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- OpenWeatherMap API

## Setup

1. Clone or download this repository
2. Get a free API key from [OpenWeatherMap](https://openweathermap.org/api)
3. Replace `"your_key"` in `script.js` with your actual API key
4. Open `index.html` in a web browser

## Usage

1. Enter a city name in the search box
2. Click the submit button
3. View the current weather information displayed below
